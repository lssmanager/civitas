Reglas visuales del sistema

No purple bias

Header claro y contenido

Altura entre 68 y 76 px, marca visible, búsqueda rápida, selector de organización y CTA primario.

Cards con tensión baja

Fondo blanco o casi blanco, borde sutil, radio corto y un solo foco visual por card.

Menús sin ruido

Dropdowns con dos niveles como máximo, icono o inicial, y acciones destructivas separadas.

Jerarquía de dashboard

Primero contexto global, luego métricas, luego tablas y actividad. Nunca una cuadrícula caótica.
